How It Works

AI, Please! uses a simple base class AIProvider that other AI SDKs or APIs extend. Each provider class implements a common method: sendPrompt(prompt).

You instantiate the library, register your providers using your API keys, and then send prompts to any registered AI by name.

import { AIPlease, OpenAIProvider } from './ai-please.js';

const ai = new AIPlease();
ai.registerProvider('openai', new OpenAIProvider({ apiKey: 'your-api-key' }));

const response = await ai.ask('openai', 'What is AI?');
console.log(response);

This abstraction lets you swap out AI providers easily without changing how you interact with them in your code.
